# VIRU Landing Page

## 🚀 How to Deploy (100% Free)

### Option 1: GitHub Pages (Recommended - Easiest)

**Steps:**
1. Create a new repository named `viru-landing` on GitHub
2. Upload `index.html` to the repo
3. Go to Settings → Pages
4. Source: Deploy from branch `main`
5. Your site will be live at: `https://rajpratham1.github.io/viru-landing`

**Time:** 5 minutes

---

### Option 2: Vercel (Most Professional)

**Steps:**
1. Go to [vercel.com](https://vercel.com)
2. Sign in with GitHub
3. Click "New Project"
4. Import the `landing-page` folder
5. Click "Deploy"
6. Your site will be live at: `https://viru.vercel.app`

**Bonus:** Free custom domain support

**Time:** 3 minutes

---

### Option 3: Netlify (Also Great)

**Steps:**
1. Go to [netlify.com](https://netlify.com)
2. Drag and drop the `landing-page` folder
3. Your site goes live instantly
4. Free at: `https://viru.netlify.app`

**Time:** 2 minutes

---

## 🎨 Customization

### Change Colors:
Edit the CSS gradients in `<style>` section:
- Primary: `#3b82f6` (blue)
- Accent: `#8b5cf6` (purple)

### Add Your Own Images:
Replace placeholder sections with screenshots:
```html
<img src="screenshot.png" alt="VIRU Demo">
```

### Update Links:
Replace `rajpratham1` with your GitHub username

## 📊 Add Analytics (Optional)

### Google Analytics (Free):
Add this before `</head>`:
```html
<!-- Google Analytics -->
<script async src="https://www.googletagmanager.com/gtag/js?id=YOUR-GA-ID"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'YOUR-GA-ID');
</script>
```

## 🌐 Custom Domain (Optional)

### Get a Free Domain:
- [Freenom](https://freenom.com) - Free .tk, .ml, .ga domains
- [GitHub Student Pack](https://education.github.com/pack) - Free .me domain

### Connect to Vercel/Netlify:
1. Buy domain or use free one
2. Add domain in Vercel/Netlify settings
3. Update DNS records (they'll guide you)

**Result:** `viru.tech` or `getviru.com`
